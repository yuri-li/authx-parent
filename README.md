# 1 简介

模拟keycloak的功能。

## 1.1 核心

权限管理的核心，就两块：认证、授权

| 类别 | 描述     | 专用名词       |
| ---- | -------- | -------------- |
| 认证 | 你是谁   | authentication |
| 授权 | 能做什么 | authorization  |
